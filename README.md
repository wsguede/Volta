# Volta

A native Android app for capturing high-quality 360° photospheres with AR-guided assistance.

Users sweep their phone in a continuous motion while the app auto-captures frames, stitches them on-device into an equirectangular JPEG, and exports it to the device photo library with GPS embedded in EXIF — ready for upload to Google Maps or Street View Studio.

See the [PRD](https://github.com/wsguede/Volta/issues/1) for full product requirements.
